# ניהול משווקים - Affiliate Program

אתר לניהול משווקים עם מעקב אחר מכירות ועמלות.

## הרצה

### אפשרות 1: Python (מומלץ)
```bash
python3 -m http.server 3000
```

### אפשרות 2: Node.js
```bash
npx http-server -p 3000
```

### אפשרות 3: PHP
```bash
php -S localhost:3000
```

לאחר ההרצה, פתח את הדפדפן בכתובת:
```
http://localhost:3000
```

## תכונות

- הוספת משווקים חדשים
- הצגת רשימת משווקים
- מעקב אחר מכירות ורווח עמלה
- מחיקת משווקים

## דרישות
